Who made this?

We are junior web devellopers of the Code Gorilla school. This is A code learning bootcamp in the Netherlands. We use entrepeneural learning and ephasise on teaching resilience to stress and fast changing working enviroments. This project is made in 3 months bij people with almost no programing skils. We hope you enjoy it.

2016

Our Humble Beginnings

Diem Do starts a project based on the idea that a lot of I.T. Busnisses are looking voor web developers and a lot of people are sitting at home looking for a meaningfull job

2017

A shool is Born

After a harsh selection procedure consisting of a impossible assignment (used to monitor our reactions) and the mandatory participation of a Hackaton (used to monitor how we are holding out under time pressure as wel as our teamworking skills) 14 people where selected for the bootkamp.

2017

The bootcamp is up and running

The bootcamp was initialy set u as a ICT. business. We managed to create something nice in the first two weeks but faild as intended to dilliver on time due to al lot of things, But we learned alot. Coping with stress, Working together, Working within a project, HTML, CSS, Javascript, Angler, Unity, Csharp etc..

2017

Making the game

We got togheter with 3 people voor the project that created this websit the game and de databas of it all. and created all you see here.
